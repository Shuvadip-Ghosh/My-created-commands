This repository has some bat or 
python(which have been converted to exe if some one dosent have python installed) file
which does some work that we can do with linux commands on windows

You can put these files into a folder and set it to path and 
you can access these command for anywhere in your file system

cat is to list files 
Usage 
cat <filename-with-extension>

github is to add commit and push changes to repository in oneline
Usage
github "<commit-message>" <remote-url-name> <branch-name>

jav to run java files from cmd
Usage
jav <filename-with-extension>

ls to list files in the format we get in linux 
ls

touch to create files from cmd
touch <filename-with-extension>
